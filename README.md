# 📄 RAG-Powered PDF Q&A Chatbot

An AI-powered Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDF documents and ask natural language questions.

The application retrieves relevant information using semantic search and generates context-aware answers using Llama 3.1.

---

## 🚀 Features

- Multi-PDF Upload
- Semantic Search with FAISS
- Hugging Face Embeddings
- Llama 3.1 via Groq API
- Chat History
- Source Citations
- PDF Analytics Dashboard
- Interactive Streamlit UI

---

## 🛠 Tech Stack

- Python
- Streamlit
- LangChain
- FAISS
- Hugging Face
- Groq API
- Llama 3.1

---

## 📊 Architecture

PDF Upload
↓
Document Loading
↓
Text Chunking
↓
Embeddings Generation
↓
FAISS Vector Store
↓
Semantic Retrieval
↓
Llama 3.1
↓
Answer Generation + Source Citations
